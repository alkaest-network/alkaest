# Alkaest

Alkaest is a decentralized protocol for coordinating work through verifiable reputation and cryptographic identities.

The protocol aims to enable open markets for services such as software development, design, automation, and real-world tasks without relying on centralized intermediaries.

Participants run nodes that manage identities, reputation, and AI-assisted task discovery.

The long-term goal is to create a universal coordination layer where reputation becomes the primary credential for economic participation.

## Status

This repository currently contains early protocol research and the initial white paper draft.

Implementation will begin after the protocol architecture and economic model are sufficiently defined.
